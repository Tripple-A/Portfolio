## Installation of This Repository

Clone the Repository,

```Shell
your@pc:~$ git clone git@github.com:Tripple-A/Portfolio.git
```

Move to the downloaded folder

```Shell
your@pc:~$ cd Portfolio
```

Open the index.html page in your browser.
